**SLAM** is the Self-service LTI App Manager for the Canvas LMS.

<img alt="Self-service LTI App Manager Tool Selection" src="https://github.com/user-attachments/assets/2176b87d-3ff0-44b7-a09a-c7f20197b202" />

It is an LTI 1.3 app forked from [Basic-LTI-PHP](https://github.com/kylejtuck/Basic-LTI-PHP) and uses the [__ceLTIc LTI class library__](https://github.com/celtic-project/LTI-PHP).

SLAM allows admins to configure LTI tools that instructors can add to their own courses via a toggle. This process allows the LMS menus to stay uncluttered, especially when you have many courses and only a few of them need a specific tool.

This is an update to an existing internal version of SLAM that takes advantage of the new Canvas Availability and Exceptions feature.

**IMPORTANT**

You should test SLAM in your beta or test environments first.

Please check the [wiki](https://github.com/kjtu228-uky/SLAM/wiki) for information about installing and configuring SLAM.

Contributions and constructive criticism welcome.
